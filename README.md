# Sözlük Karşılaştırma




&nbsp;Bu proje 2 farklı sözlük sitesinden (<a href="https://sozluk.gov.tr/">TDK Sözlüğü</a> ve <a href="https://lugatim.com/">Kubbealtı Lugatı</a>) aynı kelimenin anlamlarını karşılaştırmayı sağlıyor. Özellikle dilbilimciler, araştırmacılar ve öğrencilerin kelimelerin anlamlarındaki varyasyonları inceleyip analiz etmelerine yardımcı olacak.




Bir arama kutusu üzerinden sorgulama yaparak tek sayfada bütün anlamları görüp, kıyaslama yapabilirsiniz.
Flask ile geliştirdiğim uygulamayı  Render cloud platformunda deploy ettiğim adrese tıklayarak ücretsiz olarak kullanabilirsiniz.
İlk kullanımda sayfanın yüklenmesi 50 saniye kadar sürebilir. Bunun sebebi kullandığım Render Cloud hosting planının işleyişidir. Sonrasında kullanımınız normal hıza dönecektir. 

https://sozluk-karsilastirma.onrender.com/

- Sayfa açılışında geçen süre ilk kullanım için Render Cloud işleyişi sebeplidir.
- Kubbealtı Lügatı'nın anlam numaralandırma eksiği bulunan kelimeleri için anlam çekilme sorunu oluşabilir.
